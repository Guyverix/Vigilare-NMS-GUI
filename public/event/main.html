Existing Events

Show all existing events. There is the ability to move to history, acknoweledge, or even create tickets on events.

Replay Event

Run an event that happened before and change the mapping on how it behaves in the event system.

Remove Event

Remove a specific event from the database. This will completely remove all history of it ever happening.

Search Events

Search all events that have ever happened.

Set a Maintenance Event

Set a suppression timer for maintenance events. This is still at the todo level unfortunately.

Historical Events

Show the last several hundred "cleared" or "historical" events..
