New Maintenance Event

Setup a new maintenance event for history, or event suppression timers.

Change Existing Event

Alter details about a specific event. This can also do historical event changes, although you likely should not want to change things that have already happened.

Remove Maintenance Event

Normal use will be to remove a planned canceled maintenance event, however there is the ability to remove one that has alredy happened.

Search All Maintenance Events

This is simply a global "find all" for maintenance events. From this point you can change or cancel them.

Disable A Maintenance Event

This will likely not be used, as either it is planned or it is not. Only possibility I am seeing right now is let the event sit, but disable the event suppression. That could be done in the change event anyway so this is likely not all that useful.
