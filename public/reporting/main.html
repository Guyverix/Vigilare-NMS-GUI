New Report

Create a new report to be run on demand, or via housekeeping on a cycle. This is specific for the creation of the template itself. This will not run the template at creation time. Use Search Report Templates to run a given template when needed.

Change Report

Change a specific report which you know the ID of. This will generally be unused. Other pages will allow for this in a more consumable way. However if you DO know the id number, this will allow for changes as well.

Remove Report

Remove a specific report. This is a complete removal not a disable for a given report when the ID is known. If you make a mistake, this is not recoverable and you will have to create an enitre new report again.

Search or Run Report Templates

Search all defined templates. In general this will be for editing or running on demand a given template.

Disable Report

Disable a specific report when you know the specific reporting ID value. This is only useful against automated report generation. This will not stop a manual report run.

Purge old Reports

Remove old reports that have been run in the past.

Search Existing Completed Reports

Return a list of all completed or pending reports.


Run a defined report (unused, testing)

Choose and run a given report.
