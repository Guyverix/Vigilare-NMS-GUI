Report Management

New Report

Create a new report to be scheduled or executed on demand. This action defines the template, not the report run itself.

Change Report

Modify a report if you know the report ID. Typically used in advanced or direct-access workflows.

Remove Report

Permanently delete a report by ID. This action cannot be undone.

Search/Run Templates

Search all templates to run or edit a specific report template.

Disable Report

Disable automated generation for a specific report ID. Manual runs are unaffected.

Purge Old Reports

Clean up historical reports that were generated in the past.

Search Completed Reports

Browse completed or pending report instances.

Run Report (Test)

Select and run a defined report. Mostly for development or diagnostics.
