New Monitor

Create new monitors for NRPE, SNMP, shell, etc. This will allow for a complete new monitor. Only after creation can you add hosts or hostgroups to the monitor.

Change Monitor

Change a specific monitor which you know the ID of. This will generally be unused. Other pages will allow for this in a more consumable way. However if you DO know the id number, this will allow for changes as well.

Remove Monitor

Remove a specific monitor. This is a complete removal not a disable for a given monitor when the ID is known. If you make a mistake, this is not recoverable and you will have to create an enitre new monitor again.

Search Monitors

Search all defined monitors, both enabled and disabled that match your search criteria. From here you can choose a monitor to change.

Disable Monitor

Disable a specific monitor when you know the specific monitor ID value. This is more of a placeholder, and will be unlikely to be commonly used. The common way to do this will be to change the monitor after searching for the one you care about.

Add Host to Monitor

Add a new host to an existing monitor when you know the host id, and the monitor id values. This is also a catchall and will likely be removed. Use the search and change from there.


Add HostGroup to Monitor

Add an existing HostGroup to a monitor when you know the name of the HostGroup name and the monitor id values. This should be done in the search and change area. This is a placeholder and will be rarely if ever used.
